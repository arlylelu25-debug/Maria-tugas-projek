Analisis Kompleksitas
Input data: O(n)
Mencari nilai maksimum: O(n)
Mencari nilai minimum: O(n)
Menghitung rata-rata: O(n)
Menghitung jumlah kelulusan: O(n)
Pembuatan grafik: O(n)
Dimana n adalah jumlah data dalam array.
